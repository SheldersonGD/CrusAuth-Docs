# Repository Setup

Recommended public repository for CrusAuth documentation.

## Repository name

```text
CrusAuth-Docs
```

Alternative:

```text
crusauth-docs
```

## Description

```text
Official documentation for CrusAuth Hybrid Authentication for Paper, Folia, Velocity and BungeeCord.
```

## Website

After GitHub Pages is enabled, set the repository website to:

```text
https://YOUR_GITHUB_USERNAME.github.io/CrusAuth-Docs/
```

## Visibility

Use a public repository for documentation.

Keep the source code repository private unless you intentionally decide to open-source the project.

## Features

Recommended:

- Issues: enabled only if you want public bug reports.
- Discussions: optional.
- Wiki: disabled, because documentation lives in Markdown files.
- Projects: disabled unless you need public roadmap tracking.
- Sponsorships: optional.

## GitHub Pages

Recommended Pages source:

```text
Deploy from branch
Branch: main
Folder: /docs
```

This publishes the files inside the `docs/` folder as a documentation website.
