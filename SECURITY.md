# Security Policy

This is the public documentation repository for CrusAuth.

Do not open public issues containing exploit details, license bypass information, private endpoints, bridge secrets, license keys, database dumps, or player account data.

## Reporting a vulnerability

Send a private report through the official support channel:

- Discord: `PUT_SUPPORT_DISCORD_LINK_HERE`
- BuiltByBit: `PUT_BUILTBYBIT_RESOURCE_LINK_HERE`

Include:

- CrusAuth version;
- backend platform and version;
- proxy platform and version;
- storage type;
- exact reproduction steps;
- logs with secrets removed.

Do not include account databases, license cache files, offline license slot files, private keys, or bridge secrets.
